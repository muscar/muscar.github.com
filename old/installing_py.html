Installing Python

Windows

Download the Python installer from here (if you have a 64 bit version of windows you can use this installer). If for any reason you can't install the regular Python interpreter (e.g. yo don't have administrator rights on your Windows box) you can download a portable version of Python fro the mirrors at the bottom of this page.

Warning Avoid using spaces in the install path of your Python distribution.

Linux

Most of the current major Linux distributions already have Python installed (e.g. Ubuntu, Fedora). If you don't have a Python interpreter you can try to install it via the distribution's usual package install process (usually major distributions come with graphical package managers). If that fails you can install from source. If you're considering the last option I won't give any details since I assume you know what you're doing. Remember: 

./configure && make && sudo make install
or, if you don't have sudo installed: 
./configure && make && su -c make install

OS X

OS X comes with Python already installed.

Using the virtual machine disk image

Download this disk image. It has everything you need to get started. Download VirtualBox here (or here for Linux, and here for OS X). Install VirtualBox, then create a new virtual machine for Ubuntu Linux, set it to 512MB of RAM, and choose to use an existing disk, then navigate to the disk image you downloaded earlier. Start the virtual machine and get coding.
