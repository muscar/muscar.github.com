Negotiations

What is a negotiation?

  • a dialogue between two or more parties

  • usually taking place over multiple rounds

  • while trying to reach an agreement

ContractNet

  • it's a interaction protocol standardized by FIPA

  • it can be used to implement negotiations

  • it's already implemented in JADE ― yay!

ContractNet in JADE (I)

To implement an initiator just have your behavior extend ContractNetInitiator and override the right methods (handlePropose, handleAllResponses, handleRefuse, handleFailure, handleInform).

    
import jade.proto.ContractNetInitiator;
      
class SellerBehavior extends ContractNetInitiator {
  @Override
  protected void handleAllResponses(Vector responses, Vector acceptances) {
      ...
  }

  ...
}

ContractNet in JADE (II)

To implement participant just have your behavior extend ContractNetResponder and override the right methods (handleCfp, handleAcceptProposal, handleRejectProposal).

    
import jade.proto.ContractNetResponder;
      
class BuyerBehavior extends ContractNetResponder {
  @Override
  protected ACLMessage handleCfp(ACLMessage cfp) 
    throws NotUnderstoodException, RefuseException {
      ...
  }

  ...
}

ContractNet in JADE (III)

  • cfp: ACLMessage.CFP
  • refuse: ACLMessage.REFUSE
  • propose: ACLMessage.PROPOSE
  • reject: ACLMessage.REJECT_PROPOSAL
  • accept: ACLMessage.ACCEPT_PROPOSAL
  • failure: ACLMessage.FAILURE
  • done/result: ACLMessage.INFORM

Homework

You have a central office with 5 cabs. Use ContractNet to model the interaction between the central office and cabs to place orders.

The central office sends a CFP with x and y coordinates of the client. Cabs should replay with the distance from their own x and y coordinates to the client (use the Euclidean distance). The cab closest to the client will be selected.
